Reports Maven wrapper URLs that contain credentials or do not match canonical Maven wrapper artifact coordinates.

This inspection checks distributionUrl and wrapperUrl values in .mvn/wrapper/maven-wrapper.properties. It detects invalid URL shapes, mismatched path and file versions or artifact IDs, wrong Maven group paths, noncanonical file names, and plaintext credentials.

Quick-fixes can remove credentials, rewrite the affected coordinate segment, or replace the value with a canonical Maven Central URL when release metadata is available.
