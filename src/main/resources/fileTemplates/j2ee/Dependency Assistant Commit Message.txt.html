This built-in template is used each time Dependency Assistant commits an applied Upgrade Plan item.

This built-in template is editable. Along with plain text and Velocity directives, you can also use the predefined variables listed below. They are expanded into the corresponding values.

${DEPENDENCY} The dependency or upgrade group display name.
${FROM_VERSION} The current version, or the lowest current member version for an upgrade group.
${TO_VERSION} The planned target version.
